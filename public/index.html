Minimal example of signing and broadcasting a smart contract call to the draw method.

You must be signed in to draw. Click ‚ÄúSign In‚Äù to connect your wallet.

(open dev console to track progress)

Draw Pixel(Sign and broadcast) 

await nearWallet.sendTransaction({
  signerId: nearWallet.accountId(),
  receiverId: "berryclub.ek.near",
  actions: [
    {
      type: "FunctionCall",
      params: {
        methodName: "draw",
        args: {
          pixels: [{
            x: 10,
            y: 20,
            color: 65280, // green
          }],
        },
        gas: $$`100 Tgas`,
        deposit: "0",
      },
    },
  ],
});

Buying tokens

You can also call contract methods that require a deposit. All transactions are signed through your connected wallet.

Learn more about NEAR transactions: NEAR documentation | Transaction Flow

Buy Tokens(Requires deposit) 

await nearWallet.sendTransaction({
  signerId: nearWallet.accountId(),
  receiverId: "berryclub.ek.near",
  actions: [
    {
      type: "FunctionCall",
      params: {
        methodName: "buy_tokens",
        args: {},
        gas: $$`100 Tgas`,
        deposit: $$`0.1 NEAR`,
      },
    },
  ],
});
